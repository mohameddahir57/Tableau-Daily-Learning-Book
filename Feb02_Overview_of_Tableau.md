# Day 2: Overview of Tableau

**Date:** February 2, 2026

## 1. What is Tableau?
Tableau is a powerful and fastest growing data visualization tool used in the Business Intelligence industry. It helps in simplifying raw data into a very easily understandable format.

## 2. Key Features of Tableau
- Real-time analysis
- Data blending
- Collaboration and sharing
- No programming required

## 3. The Tableau Ecosystem
Briefly explore the different components like Tableau Desktop, Tableau Server, Tableau Online, and Tableau Public.

## Summary
Tableau provides the capability to analyze data quickly and create visualizations that are interactive and shareable.
