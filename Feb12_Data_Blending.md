# Day 12: Data Blending

**Date:** February 12, 2026

## 1. What is Data Blending?
Data blending is used when data is from two different data sources that cannot be joined directly (e.g., Excel and SQL). It happens at a sheet level.

## 2. When to Use Blending
Use blending when data sources have different levels of aggregation or are from non-relational sources.

## Summary
Data blending is a powerful way to integrate disparate data sources on the fly within a specific visualization.
