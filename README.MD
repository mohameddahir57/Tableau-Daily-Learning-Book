# Tableau Daily Learning Book (Beginner to Advanced)

## Overview
This repository contains a **daily structured learning and writing plan for Tableau**, designed to build a **complete beginner-to-advanced Tableau book**.

The goal of this project is to:
- Learn Tableau deeply and systematically
- Write **2–3 pages every day**
- Produce a final document of **35–45+ pages**
- Create strong material for **portfolio, university submission, and self-study**

Each day focuses on **one major Tableau-related topic**, explained in detail with theory, examples, and real-world use cases.

## Writing Rules
- Write **2–3 pages per day**
- One topic per day
- No chapter-based structure (Day-based only)
- Clear explanations
- Real-world examples
- Beginner-friendly but professional level
