# Day 9: Connecting Data to Tableau

**Date:** February 9, 2026

## 1. Data Connection Options
- **Excel & Text Files:** CSV, JSON, PDF.
- **Relational Databases:** SQL Server, MySQL, PostgreSQL.
- **Cloud Applications:** Google Sheets, Salesforce.

## 2. Live vs. Extract
- **Live Connection:** Real-time updates from the source.
- **Extract (.hyper):** Faster performance, snapshot of data.

## Summary
The first step of any analysis in Tableau is connecting to the relevant data source and deciding on the connection type.
